# mWin Tools

## Overview

mWin Tools is a lightweight Windows desktop application that provides quick and convenient access to over 20 Windows built-in system tools. Each tool is accompanied by an educational popup that explains its purpose and functionality before launching, making it an excellent resource for both beginners learning about Windows system administration and advanced users seeking quick access to system utilities.

Built with Rust and the egui/eframe GUI framework, mWin Tools features a modern dark-themed interface with native performance and requires no additional dependencies to run.

---

## Features

- **Extensive Tool Collection**: Access more than 20 Windows built-in system tools across multiple categories including System, Security, Maintenance, Accessibility, and Advanced tools
- **Educational Popups**: Every tool includes an informative description that explains what the tool does and when to use it
- **Modern Dark Interface**: Clean, minimalistic dark theme designed for comfortable viewing
- **Lightweight Performance**: Single executable file with native Rust performance
- **Administrator Detection**: Clearly indicates which tools require elevated administrator privileges
- **Standalone Operation**: No external dependencies required - simply download and run

---

## System Requirements

- **Operating System**: Windows 10 or Windows 11 (64-bit)
- **Permissions**: Standard user account (some tools require administrator rights)

---

## Installation

1. Download the latest installer from the Releases section
2. Run the installer (mwin-tools-setup-vX.X.X.exe)
3. Follow the on-screen installation wizard
4. Launch mWin Tools from the Start Menu or Desktop shortcut

## Included Tools

### System Tools

| Tool | Description |
|------|-------------|
| System Restore | Restore system to a previous restore point |
| System Information | View detailed hardware and software specifications |
| Services | Manage Windows background services |
| Task Scheduler | Create and manage scheduled tasks |
| Group Policy Editor | Configure local group policies (Windows Pro/Enterprise only) |
| Control Panel | Access classic Windows control panel |
| Network Connections | Manage network adapters and settings |
| Programs and Features | Install or uninstall programs |
| System Properties | View and modify system configuration |
| Display Settings | Configure display preferences and resolution |

### Maintenance Tools

| Tool | Description |
|------|-------------|
| Disk Defragmenter | Optimize hard drive performance |
| Check Disk | Scan and repair disk errors |
| Disk Cleanup | Free up disk space by removing temporary files |

### Security Tools

| Tool | Description |
|------|-------------|
| Security Center | View security status and settings |
| Windows Firewall | Configure firewall rules and settings |
| System File Checker | Scan and repair corrupted system files |
| User Accounts | Manage user accounts and passwords |

### Accessibility Tools

| Tool | Description |
|------|-------------|
| On-Screen Keyboard | Virtual keyboard input |
| Magnifier | Screen magnification utility |
| Narrator | Text-to-speech screen reader |

### Advanced Tools

| Tool | Description |
|------|-------------|
| Device Manager | Manage hardware devices and drivers |
| Registry Editor | View and edit Windows registry |
| Event Viewer | View system and application event logs |
| Command Prompt | Windows command line interpreter |
| PowerShell | Windows PowerShell scripting environment |

---

## Usage

1. Launch the application
2. Browse the tool categories using the sidebar navigation
3. Click on any tool to view its description
4. Click the "Launch" button to open the selected tool
5. For tools requiring administrator privileges, confirm the UAC prompt
